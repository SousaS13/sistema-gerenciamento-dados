# Sistema de Gerenciamento e Estruturação de Dados

Aplicação desenvolvida em ambiente acadêmico focada no processamento, organização e armazenamento seguro de informações de projetos corporativos. O sistema foi estruturado para garantir a integridade dos dados e simular cenários reais de fluxos operacionais de backend.

## Tecnologias Utilizadas
- **Linguagem Principal:** Java (Sintaxe moderna e conceitos avançados)
- **Paradigma:** Programação Orientada a Objetos (POO)
- **Banco de Dados:** Relacional (MySQL / MariaDB) e comandos estruturados SQL
- **Ferramentas:** IDE de Desenvolvimento, Git e GitHub

 ## Principais Funcionalidades Implementadas
- **Modelagem e Persistência:** Estruturação de tabelas e integração de comandos SQL para persistência, consulta rápida e organização lógica de dados estruturados.
- **Lógica Backend:** Implementação de classes, encapsulamento e herança baseados em POO para manipulação e validação rigorosa das informações de entrada.
- **Estabilidade do Sistema:** Tratamento de exceções robusto em Java para blindar a aplicação contra erros de gravação, leitura de arquivos e falhas de conexão.
